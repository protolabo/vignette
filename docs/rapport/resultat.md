# Résultats & Discussion

## Roadmap

Je vous présente ici une roadmap potentielle pour le futur du projet :

| Nom        | Difficulté     | Description                                                                                                                                                                 |
|:-----------|----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Onboarding | moyenne        | Permettre à l'utilisateur de découvrir le logiciel après son inscription pour se familiariser <br/> Ajouter du contenu démo à cette fin                                     |
| Admin      | moyenne        | Ajouter une interface admin enrichie                                                                                                                                        |
| Login      | moyenne        | Permettre le login depuis des comptes externes tels que Google                                                                                                              |
| Docker     | moyenne-dure   | Permettre le setup de tenir dans un environnement Docker                                                                                                                    |
| Langue     | moyenne-dure   | Permettre de changer la langue de l'interface                                                                                                                               |
| Wiki-like  | moyenne-dure   | Système Wikipédia-like : discussions, accréditations locales ou globales, etc. Backend déjà en place (version simple)                                                       |
| Connection | moyenne-dure + | Permettre d'établir des connections avec des établissements (universités, etc), et permettre de connecter les chercheurs et sujets de recherche                             |
| Audios     | dure           | Proposer un traitement audio et une analyse afin d'en améliorer la qualité.<br/> Eventuellement ajouter un transcripteur automatique vers Alphabet Phonétique International |
| Securité   | dure           | Renforcer et optimiser la sécurité et les accès en DB (comme ça passe par une API ...)                                                                                      |
| Video      | dure           | Ajouter la possibilité d'upload des vidéos plutôt que des images simples                                                                                                    |
| Génération | très dure      | Permettre une génération automatique et/ou assistée des images servant dans les storyboards                                                                                 |
