# Études préliminaires

Lors de dicussions avec des collègues linguistes, j'ai appris que le concept de recolter des entrées audio à partir d'
images s'appelle "Scenarimage".
Il s'agit d'une méthode connue dans le milieu des linguistes, un exemple connu
est [l'histoire de la grenouille](https://talkbank.org/childes/access/Frogs/) (frog story,
exemple [ici](http://clapi.ish-lyon.cnrs.fr/V3_Feuilleter.php?num_corpus=44)).
